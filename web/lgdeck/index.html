How to enable IP Control & generate the keycode:
  1. On the TV remote press Settings → All Settings.
  2. Navigate focus to Connection (or Network) — do not press OK yet.
  3. Quickly press 82888 on the remote.
  4. The hidden Network IP Control menu appears.
  5. Enable Network IP Control if it is off.
  6. Select Generate Keycode — note the 8-character code shown on screen.
  7. Optionally enable Wake on LAN so the TV can be powered on remotely.
  8. A new keycode can be generated at any time — the old one is immediately invalidated.
Source: WesSouza/lgtv-ip-control — TV Setup & LG IP Control Implementation Guide (v0.8, 2018)
